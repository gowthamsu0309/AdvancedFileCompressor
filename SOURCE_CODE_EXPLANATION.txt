SECTION 9 — buildComparePanel()
========================================================

Two file pickers (one per file).

  1. Reads sizes of both files
  2. Computes SHA-256 of both files via computeHash()
  3. Computes MD5 of both files

  6. Returns string to done()

SECTION 16 — CompressionService
========================================================

Static inner class. Central dispatcher for all I/O.

compress():
  - Switches on algorithm string → extension string
  - Switches again → routes to correct method
  - Times execution with System.currentTimeMillis()
  - Returns CompressionResult

compressZip(input, output, level, listener):
  - ZipOutputStream wrapping FileOutputStream
  - Calls zos.setLevel(level)
  - One ZipEntry named after the input file
  - copyWithProgress() copies bytes, fires progress callbacks

compressGzip(input, output, level, listener):
  - LeveledGZIPOutputStream (custom subclass)
  - def.setLevel(level) set in constructor
  - copyWithProgress() handles byte streaming

LeveledGZIPOutputStream:
  - Extends GZIPOutputStream
  - Calls super(out) then def.setLevel(level)
  - Exposes Deflater level (not possible with base class)

decompress():
  - Reads extension → routes to decompressZip /
    decompressGzip / HuffmanCodec / LZWCodec

decompressZip():
  - ZipInputStream → reads first ZipEntry
  - Uses entry.getSize() for progress calculation
  - Writes to file named after entry

decompressGzip():
  - Strips .gz suffix from output filename
  - GZIPInputStream → reads → writes to output
  - Fires progress callbacks

SECTION 17 — HuffmanCodec
========================================================

compress(File input, File output, ProgressListener):

  Pass 1 — Frequency Count (0%→40% progress):
    - Reads all bytes, increments freq[byte]
    - Counts distinct symbols

  Build Tree:
    - Adds each non-zero freq as HuffmanNode to PriorityQueue
    - Special case: only 1 distinct symbol → synthetic root
    - Merges two lowest-freq nodes repeatedly
    - Final single node = Huffman root

  Build Code Table (buildCodes()):
    - Recursive DFS on tree
    - Left = "0", Right = "1"
    - Leaf: stores Map<Byte, String> code
    - Edge case: single-symbol → code = "0"

  Write Output (DataOutputStream):
    - dos.writeUTF(originalFilename)  → for decompression
    - dos.writeLong(originalLength)   → stop condition
    - dos.writeInt(distinct)          → how many table entries
    - For each distinct byte: writeByte + writeLong(freq)

  Pass 2 — Encode (40%→100% progress):
    - Re-reads input file byte by byte
    - Looks up code string per byte
    - Writes each bit via BitOutputStream
    - BitOutputStream flushes padding bits at close

decompress(File input, File outDir, ProgressListener):
  - Reads header back: filename, length, freq table
  - Rebuilds identical Huffman tree (deterministic)
  - Reads bits from BitInputStream
  - Traverses tree: 0=left, 1=right, leaf=output byte
  - Stops when produced == originalLen

========================================================
SECTION 18 — LZWCodec
========================================================

compress(File input, File output, ProgressListener):
  - Writes input filename as UTF header
  - Initializes dictionary: 256 single-char → code entries
  - Greedy match loop:
      w = current string, c = next byte
      if (w+c) in dictionary → w = w+c (extend)
      else → emit code for w (16 bits), add w+c to dict
  - Auto-resets at 65,535 entries (16-bit limit)
  - Emits final w after EOF

decompress(File input, File outDir, ProgressListener):
  - Reads filename header
  - Rebuilds reverse dictionary (code → string)
  - Reads 16-bit codes from BitInputStream
  - Reconstructs strings using standard LZW decode rule:
      if code in dict → use dict[code]
      else → use w + w[0]  (deferred entry case)
  - Writes each decoded string as bytes

========================================================
SECTION 19 — BitOutputStream / BitInputStream
========================================================

BitOutputStream:
  - Wraps any OutputStream
  - writeBit(int bit):
      Shifts bit into currentByte
      Writes full byte when numBits reaches 8
  - writeBits(int value, int count):
      Writes count bits from MSB to LSB
  - flush(): zero-pads current byte to boundary
  - close(): calls flush() before stream closes

BitInputStream:
  - Wraps any InputStream
  - readBit():
      Reads new byte when buffer empty
      Returns -1 on EOF
      Extracts MSB first
  - readBits(int count):
      Accumulates count bits into result int
      Returns -1 if EOF encountered mid-read
  - close(): no-op (owning try-with-resources handles it)
